Rekonsiliasi data selalu bermuara pada satu pertanyaan yang sama: data mana yang ada di sini tapi tidak ada di sana?

Dua laporan dari sistem berbeda, dua export pada waktu berbeda, atau dua daftar ID dari tim yang berbeda — perbedaannya harus ditemukan, dan ditemukan dengan benar.

Set difference itu sederhana secara logika. Masalahnya adalah ketika pekerjaan ini dilakukan di spreadsheet, ada spasi tersembunyi.

Satu spasi di akhir nilai tidak akan terlihat di sel, tapi cukup untuk membuat COUNTIF mengembalikan nol padahal nilainya jelas ada.

Data yang seharusnya terdeteksi sebagai duplikat malah lolos — dan rekonsiliasi jadi tidak bisa dipercaya.

Alat ini dibuat untuk menyelesaikan masalah itu secara tuntas. Paste dua daftar, klik tombol, dan JavaScript akan membandingkan keduanya setelah setiap nilai dibersihkan dengan trim().

Tidak ada formula bersarang, tidak ada risiko tipe data, tidak ada false negative diam-diam.
